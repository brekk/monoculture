## groupByIndex

### Usage
```js test=true
expect(groupByIndex(0, [
  ['spades', 'ace'],
  ['diamonds', 'seven'],
  ['hearts', 'eight'],
  ['clubs', 'jack'],
  ['diamonds', 'ace'],
  ['spades', 'king'],
  ['diamonds', 'four'],
  ['spades', 'eight'],
])).toEqual({
  clubs: [['jack']],
  diamonds: [['seven'], ['ace'], ['four']],
  hearts: [['eight']],
  spades: [['ace'], ['king'], ['eight']],
})
```
