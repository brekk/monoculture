

## execWithConfig

Consume external commands as a Future-wrapped value.

### Usage
```js
import { execWithConfig } from 'kiddo'
import { fork } from 'fluture'
fork(console.warn)(console.log)(
  execWithConfig(
    function customCancellationFunction() {},
    'echo',
    { cleanup: true },
    ['ahoy']
  )
)
```
